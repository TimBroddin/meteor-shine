Meteor Trianglify
==================

[Shine.js](https://github.com/bigspaceship/shine.js) packaged for [Meteor](http://meteor.com).


### Installation

With [Meteorite](https://github.com/oortcloud/meteorite) installed:

```sh
$ mrt add shine
```

See [here](https://github.com/bigspaceship/shine.js)
